## Running this project locally

Install dependencies using **yarn** or **npm**.

Start the server

```cl
yarn dev
```

Start the web

```cl
yarn start
```

Start the mobile

```cl
expo start
```

Set config variables

API
 
 ```cl
DATABASE_URL=
```

Web

 ```cl
VITE_API_URL=
```
